New CSS Feature

The Initial-Letter

The new CSS feature; initial-letter, allows developers to more easily control and customize their drop cap. You can specify how many lines it occupies, color, position, size, and weight. There plenty of additional styling developers can do as well, including shadows and gradients. I really enjoyed using this new feature.

Here is the link to the developer blog on initial-letter: https://developer.chrome.com/blog/control-your-drop-caps-with-css-initial-letter
